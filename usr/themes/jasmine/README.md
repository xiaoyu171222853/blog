## 主题亮点

* 响应式设计
* 针对 SEO 优化
* 支持切换夜间模式
* 支持置顶文章显示
* 支持评论 QQ 头像显示
* 支持代码高亮
* 支持随机文章跳转
* 支持文章缩略图设置
* 支持外观设置备份
* 主题更新检测
* ……
